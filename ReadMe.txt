
This package downloaded from http://dev.lipidity.com/
Modified version of Alcor's QS plugin template optimized for creating an interface.
------

Installation
============

Copy to /Library/Application Support/Apple/Developer Tools/Project Templates/

In Xcode, you need to create a Source Tree that points to your Quicksilver Frameworks directory. Go to the Preferences > Source Trees section, add a new entry with the setting name as QSFrameworks. Set the path for this item to /Applications/Quicksilver.app/Contents/Frameworks/ or the equivalent.


Usage
=====


Please check http://docs.blacktree.com/quicksilver/how_to_make_plug-ins for (currently sparse) official documentation.
